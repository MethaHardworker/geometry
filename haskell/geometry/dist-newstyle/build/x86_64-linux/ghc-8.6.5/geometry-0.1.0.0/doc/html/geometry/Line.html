data Line #

The straight line of a given type, passing through two given points. The first point sets the Figures' refference point and a starting point of a line. The distance between refference points p1 and p2 sets the unit and internal scale, so that `p1 == l @<- 0` and `p2 == l @<- 1`.

extendAs :: Line -> Bounding -> Line #

The extension of a line with the same refference points.

>>> mkSegment (0, 1) `extendAs` Semibound
<Ray (0.0,0.0), 0.0°>

>>> mkSegment (0, 1) `extendAs` Unbound
<Line (0.0,0.0), 0.0°>

>>> mkRay (0, 1) `extendAs` Unbound
<Line (0.0,0.0), 0.0°>
