newtype Polyline #

Representation of a polygonal chain as a list of vertices. Vector representation of a polyline is given by the first segment of a polyline. It is used for alignment or creation of polylines.

>>> cmp $ mkTriangle [0, 1:+0, 0:+1]
1.0 :+ 0.0

>>> asCmp (1 :+ 2) :: Triangle
<Triangle (0.0,0.0) (1.0,2.0) (-1.2320508075688772,1.8660254037844393)>

>>> asCmp (1) :: Rectangle
<Rectangle (0.0,0.0) (1.0,0.0) (1.0,1.0) (0.0,1.0)>
