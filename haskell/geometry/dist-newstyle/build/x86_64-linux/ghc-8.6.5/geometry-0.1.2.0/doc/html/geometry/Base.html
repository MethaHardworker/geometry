bounds :: m -> [Double] #

Returns bounds for a parameter of a manifold. [a, b] for finite, [a] for semibound manifolds (a is lower bound), [] for unbound manifolds

paramMaybe :: m -> Double -> Maybe CN #

Returns a point on a manifold for given parameter, or Nothing if parameter runs out of the range defined for a manifold.

projectMaybe :: Affine p => m -> p -> Maybe Double #

Returns a parameter for a point on a manifold, or nothing if parameter could not be found.

param :: m -> Double -> CN #

Returns a point on a manifold for a given parameter. If parameter value is out of range [0, 1] for a finite manifolds the closest boundary value is returned.

project :: Affine p => m -> p -> Double #

Returns a parameter, corresponding to a normal point projection on a manifold. if normal projection doesn't exist returns the parameter of the closest point.

isContaining :: Affine p => m -> p -> Bool #

Returns True if point belongs to the manifold.

unit :: m -> Double #

The internal length unit of the curve, which maps the parameter to the length of the curve.

class (Figure c, Trans c, Manifold c) => Curve c where #

Class representing a curve parameterized by a real number. For finite curves parameter runs from 0 to 1, where 0 is a start and 1 is the end of the curve. The length scale is given by unit function.

All implementations must obay isomorphisms laws:

isFinite c          ==>  (project c . param c) x == x `mod'` 1
not isFinite c      ==>  (project c . param c) x == x
c `isContaining` p  ==>  (param c . project c) p == p
