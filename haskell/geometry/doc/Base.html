Curves

class Curve c where #

Class representing a curve parameterized by a real number. For finite curves parameter runs from 0 to 1, where 0 is a start and 1 is the end of the curve. The length scale is given by unit function.

All implementations must obay isomorphisms laws:

isFinite c          ==>  (project c . param c) x == x `mod'` 1
not isFinite c      ==>  (project c . param c) x == x
c `isContaining` p  ==>  (param c . project c) p == p

Minimal complete definition

(param | paramMaybe), (project | projectMaybe), (normal | tangent), distanceTo, (location | isContaining, isEnclosing)

Methods

param :: c -> Double -> CN #

Returns a point on a curve for given parameter. If parameter value is out of range [0, 1] for a finite curves the closest boundary value is returned. Use paramMaybe for more explicit parameterization.

paramMaybe :: c -> Double -> Maybe CN #

Returns a point on a curve for given parameter, or Nothing if parameter runs out of the range defined for a curve.

project :: Affine p => c -> p -> Double #

Returns a parameter, corresponding to a normal point projection on a curve. if normal projection doesn't exist returns the parameter of the closest point. Use projectMaybe for more explicit projection.

projectMaybe :: Affine p => c -> p -> Maybe Double #

Returns a parameter for a point on a curve, or nothing if parameter could not be found.

unit :: c -> Double #

The internal length unit of the curve, which maps the parameter to the length of the curve.

The distance between a curve and a point.
